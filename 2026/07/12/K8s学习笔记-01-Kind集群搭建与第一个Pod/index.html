1.1 Kind 集群搭建

什么是 Kind

Kind（Kubernetes IN Docker）把 K8s 的每个节点跑在 Docker 容器里。一台机器就能模拟多节点集群。对学习来说足够用，5 分钟起一个。

1.2 第一个 Pod

概念：Pod 是什么

Pod 是 K8s 的最小调度单位。一个 Pod 里可以有 1 个或多个容器，它们共享网络和存储。

大多数时候一个 Pod 只跑一个容器，就像 Docker 里 docker run 一样。
